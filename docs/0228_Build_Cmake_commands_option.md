# option

- 提供一个 `Boolean` 类型的选项
- 可以在 命令行中 `cmake ..` 时使用

## 用法

```
option(<variable> "<help_text>" [value])
```
- 默认值 `OFF`

## Reference

- [Documentation->cmake-commands(7)->option](https://cmake.org/cmake/help/latest/command/option.html)
