# NUMA 架构

- NUMA - `Non Uniform Memory Access/非统一内存访问`
  - SMP - `symmetric multiprocessor` - 对称处理器
- Node - [CPU, Memory]
- Optimization
  - 核绑定
    - Core 0 是系统调度核，避免绑定
  - 2 CPU/2 Instance
  - 线程数和物理核数一致

## UseCase
- Tools
