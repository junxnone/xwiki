## IPython Magics
执行Magics时，事实上是调用了该Magics定义的一个函数
- Line Magics（一个%），传入函数的是当前行的代码
- Cell Magics（两个%），传入的是整个Cell的内容


## IPython Widgets
IPython Widgets是一种基于Jupyter Notebook和IPython的可交互控件
