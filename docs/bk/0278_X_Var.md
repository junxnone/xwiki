# 变量命名法则

## 下划线

  - `first_second_name`

## Pascal 帕斯卡拼写法

  - 匈牙利命名法
  - `DisplayInfo`

## 驼峰法

  - 第一个单词小写·,从第二个单词开始每个单词首字母大写
  - `firstSecondThxxxx`

## 匈牙利命名法

  - 属性 + 类型 + 对象描述
