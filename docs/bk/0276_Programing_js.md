  - 值类型(基本类型)
      - Boolean(`true`/`false`)
      - Number
      - String
      - Null(设置为 `null`, 清空变量)
      - Undefined(对象不含值)
      - Symbol
  - 引用数据类型(对象类型)
      - Array
      - Object
      - Function
      - Date
      - Regex
      - ......

  - Number - `34`/`34.0`/`34e-5`
