# intel-telemetry-tool

  - CPU/GPU evaluate calculation capability Tool Kit

## Build & Install

    git clone https://github.com/Xiaogang-Li/intel-telemetry-tool.git
    cd intel-telemetry-tool/build
    ./prebuild.sh
    cd ..
    ./build.sh
    sudo cp telemetry /usr/bin/

## UseCase

    sudo telemetry

  - s : show statistics
