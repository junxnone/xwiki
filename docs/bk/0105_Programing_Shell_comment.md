## Brief

  - 单行注释

<!-- end list -->

    # this is comment test

  - 多行注释

<!-- end list -->

    :<<eof
    this is comment
    this is comment
    this is comment
    this is comment
    eof

    :<<!
    this is comment
    this is comment
    this is comment
    this is comment
    !

    : '
    this is comment
    this is comment
    this is comment
    this is comment
    '

<kbd>:</kbd> & <kbd>'</kbd> 之间有空格

  - shell script 中 `\` 换行的注释 \` \# xxxx\`

<!-- end list -->

    python3 preprocess.py \
       `# -input filename`\
        -input filename\
        -input filename\

    python3 preprocess.py \
        -input filename `# xxxx` \
        -input filename\
        -input filename\
