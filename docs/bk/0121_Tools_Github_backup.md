## github-backup

  - **Install github-backup**

<!-- end list -->

    pip install github-backup

  - **Backup all your github**

<!-- end list -->

    github-backup your_username -t your_token  --all -o backup

> token 勾选 repo 选项，使 其可以读取私有repo
