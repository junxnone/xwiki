# hydra

  - 由 facebook 开源的 python 动态配置框架
  - 可以分别通过 config file 和命令行配置参数
  - 可以通过目录 创建 `config group` 创建不同的配置
  - 可以通过单个命令运行不同参数的任务

> The name Hydra comes from its ability to run multiple similar jobs -
> much like a Hydra with multiple heads
