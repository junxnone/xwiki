# Intel iGPU

  - Integrated Graphics Processing Unit - 集成内置 GPU - 核显
  - 主打低功耗、低成本、高集成度，是轻薄本、办公机、迷你主机的标配，近年性能大幅提升，可覆盖办公、视频、轻度游戏与 AI 推理。
  - 无独立显存，共享系统内存；功耗低（几瓦～几十瓦）、成本低、散热简单；性能低于同代独显，但满足日常与轻创作。
  - **优势**：
      - 整机更轻薄、续航更长（笔记本）
      - 无独显故障风险，可作为备用显示输出
      - 视频编解码（Quick Sync）、AI 推理效率高
      - 无需额外购买显卡，降低装机成本
