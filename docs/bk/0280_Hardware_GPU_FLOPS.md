# FLOPS

  - `Floating-Point Operations Per Second` - 每秒浮点运算次数
  - MFLOPS - `megaFLOPS` - $10^6$
  - GFLOPS - `gigaFLOPS` - $10^9$
  - TFLOPS - `teraFLOPS` - $10^{12}$
  - PFLOPS - `petaFLOPS` - $10^{15}$
  - EFLOPS - `exaFLOPS` - $10^{18}$
