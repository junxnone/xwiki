# Linux 程序性能分析工具

  - 计数器类型工具
      - vmstat 内存统计工具
      - iostat 磁盘统计工具
      - top 处理器统计工具
      - ps 进程统计工具
  - 跟踪工具(默认不开启)
      - gdb 跟踪调试工具
      - pstack 跟踪进程栈
      - strace 跟踪系统调用和信号
  - 监视工具 统计性能运行数据
      - netstat 网络监控工具
      - sar
      - iotop 硬盘监控工具
      - mpstat 实时系统监控工具
  - 刨析工具
      - gprof
      - perf
      - nvprof - cuda
      - vtune
