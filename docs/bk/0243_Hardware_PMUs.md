# PMUs

  - PMUs - Performance Monitoring Units - 性能监控单元
  - PMUs 是 CPU 内核中的专用逻辑块，用于记录计算系统上发生的特定硬件事件。
      - 例如 `Cache Misses` / `Branch Mispredictions`
  - 根据时间创建一些高级指标, 例如 `CPI`
