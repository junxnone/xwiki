# Intel Vtune iGPU Analysis

## Setup
- Intel Metric Discovery API Library
  - Windows 集成在 Drivers 里面
  - [Linux Install](https://github.com/intel/metrics-discovery)
- Enable Permissions
- Enable GPU utilization events (i915 ftrace events)
