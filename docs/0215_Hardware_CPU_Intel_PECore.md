# P-Core & E-Core

- **P-Core** `Performance-cores` 为性能核心, 专门负责强化单线程处理能力，在游戏、生产力创作中都可以提供超强性能
- **E-Core** `Efficient-cores` 为能效核心, 主要加强多线程处理能力，可以应对大吞吐量负载，并针对后台应用进行优化。
