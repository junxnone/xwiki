# API Vs ABI

- **API** `Application Program Interface` 
  - **描述内容**: 功能函数接口
  - **层级**: 代码模块层级
  - **兼容性**: 兼容难度高
- **ABI** `Application Binary Interface` 
  - **描述内容**: 二进制文件格式/内容/...
  - **层级**: 二级制层级
  - **兼容**: 容易
