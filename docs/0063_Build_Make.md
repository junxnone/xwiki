## Brief
- **make tools**: 不同的工具 `Makefile` 格式也不同
  - **GNU Make**
  - QT qmake
  - MS nmake
  - BSD pmake
- [[cmake]] 可以做到 `Write Once, Run Everywhere`





## UseCase

- list all target

```
list:
    @grep '^[^#[:space:]].*:' Makefile
```
```
make list
```
