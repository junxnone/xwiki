# STL Standard Template Library
- 最初由惠普实验室开发，于 1998 年被定为国际标准，成为 C++ 程序库的重要组成部分
- **内置**到 `C++ Compiler` 中(头文件方式提供)
- 包含一些容器/算法/...


## 基本组件
- **容器(Containers)**: 封装有数据结构的模板类
  - 序列容器
  - 关联容器
  - 容器适配器
- 算法
- 迭代器
- 函数对象
- 适配器
- 内存分配器
