# Level Zero
- 一个接口规范，用于直接访问加速器(GPU/FPGA/..)




## Arch

- Device discovery
- Memory allocation
- Peer-to-peer communication
- Inter-process sharing
- Kernel submission
- Asynchronous execution and scheduling
- Synchronization primitives
- Metrics reporting
