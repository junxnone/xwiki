# CPU History
- 1947年12月16日，晶体管之父––威廉·肖克利和约翰·巴顿，沃特·布拉顿在贝尔实验室共同研制出世界上第一块晶体管
- 1957年仙童半导体公司成立
- 1968年诺伊斯，摩尔和格鲁夫离开了仙童半导体公司，创立了Intel公司
- 1969年，仙童半导体公司的另外七名员工创办了AMD公司
- 1971年11月15日，英特尔公司造出了世界上第一块CPU––4004微处理器 4bit, 10nm
