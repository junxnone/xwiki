## html 表示 

### 无序 List

```
<ul><li>L1</li><li>L2<ul><li>L2.1<ul><li>L2.1.1</li></ul></li></ul></li><li>L3</li></ul>
```

<ul><li>L1</li><li>L2<ul><li>L2.1<ul><li>L2.1.1</li></ul></li></ul></li><li>L3</li></ul>

### 有序 List

```
<ol><li>L1</li><li>L2<ol><li>L2.1<ol><li>L2.1.1</li></ol></li></ol></li><li>L3</li></ol>
```

<ol><li>L1</li><li>L2<ol><li>L2.1<ol><li>L2.1.1</li></ol></li></ol></li><li>L3</li></ol>


### 自定义 List

```
<ol type="I"><li>第一行</li><li>第二行</li><li>第三行</li></ol>
```

<ol type="I"><li>第一行</li><li>第二行</li><li>第三行</li></ol>



- **type:**
  - a: 小写字母
  - A: 大写字母
  - i: 小写罗马数字
  - I: 大写罗马数字
  - 1: 数字
