# 比较 branch/commit
- 在分支、标记、提交、复刻和日期之间比较仓库的状态。
- `repoUrl/` + `compare` : 手动选择想要对比的branch, 也可以比较 frok 之后的 repo
- `repoUrl/` + `compare/` + `branch_1...branch_2`|`commit_1...commit_2`
- https://github.com/junxnone/xwiki/compare
