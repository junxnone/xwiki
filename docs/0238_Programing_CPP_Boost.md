# Boost
- 1998 ~ 
- C++ 标准库的重要来源
- 大部分实现在头文件中，不需要预编译 [*需要编译的部分*](https://www.boost.org/doc/libs/1_81_0/more/getting_started/windows.html#header-only-libraries)

## 包含内容

- 字符串与文本处理
- 容器
- 迭代器
- 算法
- 图像处理
- 模板元编程
- 并发编程
- 高级网络通信
- 时间处理
- [.........](https://www.boost.org/doc/libs/1_81_0/libs/libraries.htm#Category)
